ClassroomIO is open source and we welcome contributions. These guides cover getting the project
running on your machine, the conventions we follow, and the infrastructure behind the platform.

If you are here to *use* ClassroomIO rather than work on it, start with
[What is ClassroomIO](/guides/what-is-classroomio) instead.
