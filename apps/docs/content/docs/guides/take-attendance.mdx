---
title: How to Take Attendance
description: Record lesson attendance for each student in a course using the Attendance tab.
---

The **Attendance** tab in a course lets you track which students attended each lesson. Only students who are enrolled in the course appear in the attendance grid — enroll them first via **People** if they are missing. It displays a grid with students as rows and lessons as columns. A checked box means the student was present; unchecked means absent.

## Mark attendance

From the course dashboard, open the **Attendance** tab.

- Check the box at the intersection of a student and a lesson to mark them **Present**.
- Uncheck the box to mark them **Absent**.

Changes save immediately — there is no separate save button.

Use the search bar to filter students by name if the course has a large group.

:::info[Students see read-only attendance]
Students can view their own attendance record in the Attendance tab when they open the course in the LMS, but they cannot edit it.
:::

## Pagination

The table shows 10 students per page. Use the page controls at the bottom to move between pages.
